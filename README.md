# 🍽️ Food Orders Data Analysis Project 
### Python + Power BI Integration Project 

## 📌 Project Overview
This project demonstrates an end-to-end Business Intelligence workflow by integrating **Python** with **Power BI**.

- Data Cleaning performed using Python 
- Exploratory Data Analysis (EDA) performed using Python 
- Data Visualization and Dashboard creation done in Power BI 

---

## 🎯 Project Objective
To analyze food order data and generate business insights such as:

- Number of Orders per City 
- Number of Orders per Quarter 
- Number of Cancelled Orders 
- Number of Orders per Restaurant 

---

## 🛠️ Tools & Technologies Used

- Python 
- Pandas 
- NumPy 
- Matplotlib 
- Jupyter Notebook 
- Power BI 
- CSV Dataset 

---

## 🧹 Data Cleaning Steps (Python)

- Handled missing values 
- Removed duplicate records 
- Converted date column into datetime format 
- Created a Quarter column 
- Standardized column names 

---

## 📊 Exploratory Data Analysis (EDA)

### 1️⃣ Orders per City
Used `groupby()` to calculate total orders for each city.

### 2️⃣ Orders per Quarter
Extracted quarter from order date and analyzed business performance.

### 3️⃣ Cancelled Orders Analysis
Filtered cancelled status and calculated total cancellations.

### 4️⃣ Orders per Restaurant
Calculated total orders handled by each restaurant.

---

## 📈 Power BI Dashboard Features

- Bar Charts 
- Pie Charts 
- KPI Cards 
- Slicers (Filters) 
- Interactive Dashboard 

---

## 🔍 Key Insights

- Identified high-demand cities 
- Found peak performing quarters 
- Analyzed cancellation trends 
- Identified top-performing restaurants 

---

## 🚀 Conclusion

Python helped in data cleaning and exploratory analysis, while Power BI enabled interactive visualization and business reporting.

This project demonstrates practical implementation of Business Intelligence workflow.
